What is?

It's msvcore - crossplatform library C++. Version 2. First build in 2005.

Build on: Windows, Linux, Android(Jni), flash(Crossbridge).

How to use it

* Create new c++ project.
* Create new .cpp file.
* Add code to this file: 

#define PROJECTNAME "projectname" // Set your project name
#define PROJECTVER PROJECTNAME ## _versions
#define USEMSV_GENERALCPP

// Add msvcore v.2
#include "PATH_TO_MSVCORE/msvcore2/msvcore.cpp" // Set your path

// You may use prject versions
Versions PROJECTVER[]={
	// new version to up
	"0.0.0.1", "01.12.2016 15:51",
	"0.0.0.0", "23.11.2016 10:45"
};

// Main
int main(int args, char* arg[]){
	msvcoremain(args, arg);
	print(PROJECTNAME, " v.", PROJECTVER[0].ver," (", PROJECTVER[0].date, ").\r\n");

	// Write your code
	
	return 0;
}


* Ok. It's simple project.


Base info


Null-terminate string

Please, don't use null-terminate string. Use VString[BAD LINK]. It have unsigned char * data - pointer to data, and unsigned int sz - size. It's sefe. If char * string not contained null - ooops, program may crash. VString[BAD LINK] have size, programm don't crash where string have bad data.


Classes


Core

Versions - project versions.
MsvCoreState - process state. (Auto, don't use.)
ThreadStringCore - Helper for TString[BAD LINK] class. (Auto, don't use.)
MsvCoreActivateSocket - initialize socket. Call WSAStartup() for Windows, signal(SIGPIPE, SIG_IGN) for linux. (Auto, don't use.)

String

VString[BAD LINK] - Virtual String. Use as string pointer.
MString[BAD LINK] - String. Use as class value.
SString[BAD LINK] - Stack String. Use as temporary.
TString[BAD LINK] - Thread / Temp String use as temporary.
LString[BAD LINK] - Line string. Use for adding strings.
MStringS[BAD LINK] - Short String use for conver ASCII(char) string to utf(short).

Allocate

AListAllocDef - malloc() / free() allocator.
AListAllocUList - Unlimited List allocator.
AListAllocOList[BAD LINK] - Object List allocator. Best.
AListAllocOListFast[BAD LINK] - Object List allocator. Need very more memory.
AListAllocOListTree[BAD LINK] - Object List allocator used Tree. So slow.

Dynamic arrays

IList[BAD LINK] - Ideal List. Vector. One memory block.
OList[BAD LINK] - Object List. Used AListAllocOList[BAD LINK].
UList[BAD LINK] - Unlimited List. Used AListAllocUList.
TrieList[BAD LINK] - Trie dynamic array.

Config

ConfLine[BAD LINK] - Configuration file class. Can Get()/Set() key/value.
ConfLineOptions[BAD LINK] - Configuration class. Can only Get() key/value.

Run process

PipeLine[BAD LINK] - exec process using LString[BAD LINK] bufer.
PipeLine2[BAD LINK] - exec process using ring buffer.

Regular extensions

MSVEF[BAD LINK] - my variant of regfular extension.

Hash

Base64[BAD LINK] - encode & decode to base64.
MHash[BAD LINK] - md5, sha1, sha256, rmd1 hash functions.

Http

MCookie[BAD LINK] - Http cookie container. Cookie array. Generate cookie request.
MHttp[BAD LINK] - Http head class.
GetHttp[BAD LINK] - Http[s] request.
GetHttp2[BAD LINK] - Http[s] request used MCookie[BAD LINK].

Net

ConIp[BAD LINK] - Connect to ip used ip v4 adress. Or listen port. Return SOCKET.

Crypto

MsvCoreActivateSsl - initialize SSL. (Auto, don't use.)
MsspCoreEl - class for testing & find errors.
MySSL[BAD LINK] - OpenSSL helper for accept, connect, recv, send functions.

Buffer

RecvData[BAD LINK] - one memory buffer for readed data.
SendData[BAD LINK] - memory buffers.
SendDataRing[BAD LINK] - ring memory buffer.

Io

ILink[BAD LINK] - Parser web and file link.
Readdir[BAD LINK] - read dir functions.

Lock

GLock[BAD LINK] - Critical section. Disable double lock, it's deadlock.
TLock[BAD LINK] - Critical section. Enable multilock in one thread.
CLock[BAD LINK] - Condition lock. Lock to signal.
ALock[BAD LINK] - Auto lock/unlock Glock & Tlock.

MultiType

MultiType[BAD LINK]
MTEDTX[BAD LINK]
MTEV[BAD LINK]
MTEVS[BAD LINK]
MTEVA[BAD LINK]

Time

MTime[BAD LINK] - Time functions.
MTimer[BAD LINK] - Timer functions.
